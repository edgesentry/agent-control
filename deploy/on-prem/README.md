# On-prem / air-gapped lab bundle

Edge and on-prem deployment is the **primary** shape for Cap Vista trials: Guardian + Observed Agents run on evaluator hardware without a cloud control plane.

## Quickstart (bare metal / VM)

From the repository root on your target host:

```bash
cargo build --workspace --release
./target/release/lab
./target/release/soc
```

Policy packs (once added in issue #6) load from `policies/p0/` and `policies/soc/` relative to the working directory.

## Air-gapped dependency vendoring

On a connected build machine:

```bash
cargo vendor vendor/
```

Copy the repository **including** `vendor/` and `.cargo/config.toml` (generated by `cargo vendor`) to the isolated host, then:

```bash
cargo build --workspace --release --offline
```

## Hardware reference

Submission demos target **RPi5-class or evaluator VM** (2+ vCPU, 4 GiB RAM). No GPU required for the ACS control plane.
